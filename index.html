Core Capabilities

Engineered to
pass every filter.

Built on the exact architecture that modern Applicant Tracking Systems expect to see.

Privacy First Architecture
100% client-side. No database, no login, no server. Your personal career data never leaves your browser — ever.
Live Synchronized Preview
React 18 state binds directly to the resume render tree. Every keystroke updates the document in real-time, zero latency.
Hybrid Browser Exports
High-fidelity PDF via html2canvas + jsPDF. Native .docx via pure XML packed with JSZip — no plugins needed.
ATS Single-Column Logic
No tables, no columns, no icons — just clean, hierarchical HTML that parsing engines digest perfectly every time.
Precision Python Utilities
bump_font.py and bump_font_small.py surgically re-scale CSS pt values and XML half-point nodes for pixel-perfect page fitting.
Glassmorphism UI
Premium loading overlays, CSS-driven micro-interactions, pop-and-pulse buttons — a high-end experience worthy of your application.
The Process

From blank page
to hired.

01
Fill the Form
Enter your name, title, experience, education, and skills into the structured input panels. The UI guides you through every required ATS field.
02
Watch It Render Live
As you type, the right-side preview updates instantly. Locked in real-time React sync — what you see is exactly what the parser will read.
03
Fine-tune with Python
Need to squeeze everything onto one page? Run bump_font_down.py to reduce all font sizes by 2.5pt across CSS and XML in one command.
04
Export & Apply
Hit Export PDF for a pixel-perfect visual, or Export .docx for a natively editable Word document. Both generated instantly in your browser.
